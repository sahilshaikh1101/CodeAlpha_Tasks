# CodeAlpha DevOps Internship Tasks

## Overview

This repository contains the projects completed during the CodeAlpha DevOps Internship. The tasks focus on practical implementation of DevOps concepts including containerization, build automation, Linux administration, and version control.

## Completed Tasks

### Docker Web Server

Developed and deployed a web server using Docker and Nginx. The project demonstrates containerization, image creation, container management, and web server deployment.

**Technologies:** Docker, Nginx, HTML, RHEL 9

### Java Application using Gradle

Built and executed a Java application using Gradle. The project demonstrates build automation, project management, and application execution using Gradle.

**Technologies:** Java 17, Gradle, RHEL 9
